# Brand Analysis on BlueSky: Ryanair vs Wizz Air
## Overview
Comparative social media brand analysis of Ryanair and Wizz Air on BlueSky.
Covers sentiment analysis, engagement metrics, and brand perception.

## Structure
- `report/` – Final PDF report
- `notebooks/` – Jupyter analysis notebook
- `data/` – Raw/processed data (gitignored)

## Key Methods
- Sentiment analysis (VADER / TextBlob)
- Engagement & reach metrics
- Word clouds, bar charts, time-series visualisations
